# React + TypeScript + Vite

# Tienda Shoes backen

1. clonar el proyecto
2. la base corre en SQLite y se crea sola en `backend/data/tienda_shoes.db`
3. clonar el archivo `.env.template` y renombrarlo a `.env`
4. levantar el servidor de desarrollo con `npm run dev`
5. verificar `GET http://localhost:3000/api/health`
6. probar `GET http://localhost:3000/api/store`
